ToDoListNg Angular, TypeScript, SCSS, AWS

ToDoListNg is an web application for creating, reading, updating, and deleting tasks. The application is integrated with AWS cloud services such as DynamoDB for task storage and AWS Cognito for user authentication. The backend API was created with Amazon API Gateway. The backend API communicates with AWS Lambda functions to handle DB queries.

Val-Info Ionic, Vue 3, TypeScript, Tailwind CSS

Val-Info is an Ionic/Vue mobile web app orientent around the video game, Valorant. The app features patch notes, a news feed and a lineup database.

Virtual Synthesizer Angular, TypeScript

A synthesizer web application that can be played using a keyboard or with a touchscreen. The user can enable oscillators and modify their waveform, octave, gain, tuning. Filters can also be applied to the oscillators.

Mobile Virtual Synthesizer Angular, Ionic, TypeScript, SCSS

Ionic mobile web application port of Virtual Synthesizer. Synthesizer can be played with a touchscreen. The user can enable oscillators and modify their waveform, octave, gain, tuning. Filters can also be applied to the oscillators.

3D Animated Cones Vue 3, Three.js, TypeScript, Tailwind CSS

A small Three.js web app that randomly generates cones with animated rotations. The user may add randomly generated cones or cones with custom attributes (radius, height, radial segments, color, position). Created mainly to practice working with Vue.js and Three.js.

CA COVID-19 Dashboard Python 3.7, Streamlit, Pandas, GeoPandas, pydeck, Altair

Dashboard was created using the streamlit library.

The project encompassed utilizing the pandas library for data cleaning, formatting, aggregation, and analysis. Geographical data was formatted with the geopandas library and visualized using the pydeck library. Charts were visualized using the altair library.

Dashboard contains vaccination data broken down by demographics, vaccine maker, and by county. County data is displayed on a pydeck 3D map.

Pathfinding Algorithm Visualizer Python 3.7, PyQt5

Pathfinding Algorithm Visualizer is a desktop application built with Python.

On this app, you can modify a grid by adding and removing barriers. There are three different types of barriers that at your disposal. Two of which (the green and orange blocks in the image below) can have their weight be modified. The black barriers, however, are impenetrable.

You can also specify the starting and ending locations for the path, whether to animate the pathfinding algorithm and which algorithm to utilize.

Supported algorithms: Breadth-first search, Dijkstra, A-Star.
