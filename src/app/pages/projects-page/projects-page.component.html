ToDoListNg Angular, TypeScript, SCSS, AWS

ToDoListNg is an web application for creating, reading, updating, and deleting tasks. The application is integrated with AWS cloud services such as DynamoDB for task storage and AWS Cognito for user authentication. The backend API was created with Amazon API Gateway. The backend API communicates with AWS Lambda functions to handle DB operations.

ValInformer (PWA) Ionic, Vue 3, TypeScript, Tailwind CSS

ValInformer is an Ionic/Vue progressive web app orientented around the video game, Valorant. The app features patch notes, development updates, a news feed and a lineup database.

A bit about the backend:

  • Statically hosted files are encoded (brotli) to ensure the smallest amount of data is transferred.
  • Images are programmatically retrieved depending on the device's screen size ensuring the images are sufficiently large.
  • An API Gateway HTTP API was created to circumvent restrictive CORS policies. The API invokes an AWS Lambda function which fetches an RSS feed from a URL (that is passed as a parameter) and returns the feed in the API response.
Digit Classifier API Flask, Python 3, Scikit-learn, Vue3, TypeScript

K nearest neighbors classification and random forest classification models were trained using the MNIST digits classification dataset.

The models were implemented in python 3 using the scikit-learn library. In particular, the KNeighborsClassifier and RandomForestClassifier classes were used.

These two models were chosen since they were the best performing (the highest scores against the test dataset) out of the four models tested. Afterwards, the two models' hyperparameters were fined tuned using the GridSearchCV class.


To deploy the models, I created an API with flask that receives images and returns the model's prediction and probabilities.

I then developed a frontend with Vue.js to interface with the API. The frontend has a drawing canvas where the user can draw a digit and request the API for a prediction.

CA COVID-19 Dashboard Python 3.7, Streamlit, Pandas, GeoPandas, pydeck, Altair

Dashboard was created using the streamlit library.

The project encompassed utilizing the pandas library for data cleaning, formatting, aggregation, and analysis. Geographical data was formatted with the geopandas library and visualized using the pydeck library. Charts were visualized using the altair library.

Dashboard contains vaccination data broken down by demographics, vaccine maker, and by county. County data is displayed on a pydeck 3D map.

Pathfinding Algorithm Visualizer Python 3.7, PyQt5

Pathfinding Algorithm Visualizer is a desktop application built with Python.

In this app, you can modify a grid by adding and removing barriers. There are three different types of barriers that at your disposal. Two of which (the green and orange blocks in the image below) can have their weight be modified. The black barriers, however, are impenetrable.

You can also specify the starting and ending locations for the path, whether to animate the pathfinding algorithm and which algorithm to utilize.

Supported algorithms: Breadth-first search, Dijkstra, A-Star.
